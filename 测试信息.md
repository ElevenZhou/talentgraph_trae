## 📝 测试用例

### 1. 用户登录
- 输入正确邮箱密码 → 成功登录
- 输入错误密码 → 显示错误提示
- 未登录访问受保护页面 → 重定向到登录页

### 2. 简历转换
- 上传 PDF 文件 → 正确提取文本
- 粘贴文本内容 → 正确解析
- 点击解析按钮 → 显示加载状态

### 3. 人才图谱
- 解析完成后 → 可查看可视化图谱
- 点击节点 → 显示详细信息

### 4. 运营后台
- 访问仪表盘 → 显示统计数据
- 查看用户列表 → 支持搜索筛选
- 查看简历列表 → 支持状态管理
