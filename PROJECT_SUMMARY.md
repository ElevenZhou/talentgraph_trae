   - Next.js 14 + TypeScript（现代前端）
   - SQLite（本地数据合规）

   - D3.js（可视化）
   - 架构清晰，代码规范

   - 从用户注册 → 简历解析 → 图谱可视化 → 智能匹配，全流程贯通

   - 有完整的 API 设计

   - 使用本地 SQLite，数据不出境
   - 符合国内数据法规要求
   - Token 认证机制安全

5. **代码质量**
   - TypeScript 严格类型检查
